# Practica 6

# Práctica de Autómatas de Pila (PDA) 

## Introducción

Este repositorio contiene el desarrollo de la práctica de autómatas enfocado en la **Practica 2**, donde originalmente se trabajó con **AFD y AFN**, y posteriormente se realizó su conversión e implementación utilizando **Autómatas de Pila (PDA)** mediante el software **JFLAP 7.1**.

Además, se incluye una aplicación con interfaz gráfica capaz de trabajar con los autómatas desarrollados y soportar operaciones relacionadas con autómatas de pila.

---

## Objetivo

El objetivo de esta práctica es:

* Diseñar autómatas de pila utilizando JFLAP
* Comprender el funcionamiento de la pila en los PDA
* Implementar operaciones de apilar y desapilar
* Validar cadenas de entrada utilizando autómatas de pila
* Implementar una aplicación capaz de soportar y visualizar PDA

## Código fuente

El repositorio contiene el código de una aplicación con interfaz gráfica que permite:

* Cargar autómatas

El código se encuentra organizado en carpetas para facilitar su comprensión y mantenimiento.
